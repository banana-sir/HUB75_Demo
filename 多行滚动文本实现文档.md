## 概述

多行滚动文本功能允许在HUB75 LED矩阵上同时显示多行不同字体大小的滚动文本。通过将每行的信息独立存储，实现了不同字体大小的文本互不干扰地同时滚动。

### 关键特性
- ✅ 支持多行同时滚动（最多PANEL_RES_Y/16行）
- ✅ 每行独立的字体大小
- ✅ 每行独立的滚动速度
- ✅ 每行独立的滚动方向
- ✅ 每行独立的文本颜色
- ✅ 自动检测并清除重叠文本
- ✅ 动态适配不同屏幕分辨率

## 核心设计

### ScrollLine结构体

每行滚动文本由`ScrollLine`结构体独立管理：

```cpp
struct ScrollLine {
    char *content;              // 滚动文本内容（动态分配）
    int xPosition;              // 当前X坐标位置
    int yPosition;              // 行的Y坐标（固定）
    bool isScrolling;           // 是否正在滚动
    bool isActive;              // 该行是否激活
    int scrollSpeed;            // 该行的滚动速度（1=慢，2=中，3=快）
    int scrollXMove;            // 该行每次滚动移动的像素数
    int scrollTimeDelay;        // 该行的滚动时间间隔
    ScrollDirection scrollDirection;  // 滚动方向
    uint16_t color;             // 该行的文本颜色
    int textSize;               // 该行的字体大小（独立记录）
    unsigned long lastUpdateTime;  // 该行上次更新的时间
};
```

### 核心设计思想

1. **解耦全局与局部**: `textSize`全局变量控制静态文本和新滚动文本的字体大小，`ScrollLine.textSize`记录每行的实际字体大小
2. **独立状态**: 每行维护自己的滚动状态，互不干扰
3. **动态内存**: 使用`malloc/free`管理文本内容，支持灵活的文本长度
4. **重叠检测**: 在添加新文本时检测并清除重叠的现有文本

## Y坐标系统

### Y坐标计算规则

滚动文本的Y坐标固定为 `(line - 1) * 16`，不随字体大小变化：

| 行号(line) | Y坐标 | 说明 |
|-----------|-------|------|
| 1 | 0 | 第1行，从y=0开始 |
| 2 | 16 | 第2行，从y=16开始 |
| 3 | 32 | 第3行，从y=32开始 |
| 4 | 48 | 第4行，从y=48开始 |

### 设计原因

- **简化坐标系统**: 每行有固定的起始Y坐标，便于计算和重叠检测
- **支持多字体共存**: 不同字体大小的文本可以共享同一行系统
- **统一渲染逻辑**: 在`loop()`中动态切换字体大小渲染不同行

### 代码实现

```cpp
// DisplayManager.cpp: displayText()
int yPosition = (line - 1) * 16;
scrollLines[index].yPosition = yPosition;
```

### 字体高度与覆盖关系

不同字体大小的高度：

| 字体大小 | 高度（像素） | 行数范围（64像素高度） |
|---------|------------|---------------------|
| 1 | 16 | 占1行 |
| 2 | 32 | 占2行 |
| 3 | 48 | 占3行 |
| 4 | 64 | 占4行（全屏）|

例如：
- `line=2, textSize=1`: 渲染区域 y=16~32
- `line=2, textSize=2`: 渲染区域 y=16~48，会覆盖line=3

## 最大行数计算

### 计算公式

```
maxLines = PANEL_RES_Y / 16
```

### 不同分辨率的支持

| 分辨率 | maxLines | 说明 |
|--------|----------|------|
| 64x32 | 2 | 支持2行滚动文本 |
| 64x64 | 4 | 支持4行滚动文本 |
| 64x128 | 8 | 支持8行滚动文本 |

### 代码实现

```cpp
// DisplayManager.cpp: 构造函数
DisplayManager::DisplayManager() :
    maxLines(PANEL_RES_Y / 16),
    // ...
{ }

// DisplayManager.h: 声明
int maxLines;  // 最大行数（由PANEL_RES_Y/16计算得出）
```

### 优势

- **自动适配**: 修改PANEL_RES_Y后自动计算最大行数
- **无需硬编码**: 不需要手动设置maxLines常量
- **灵活扩展**: 支持各种分辨率的面板

## 字体大小共存

### 问题场景

如果先设置字体大小为1显示滚动文本，再设置字体大小为2显示另一行滚动文本，如果不做特殊处理，第一行文本会静止。

### 解决方案

#### 1. 每行独立记录字体大小

```cpp
struct ScrollLine {
    int textSize;  // 该行的字体大小（独立记录）
    // ...
};
```

#### 2. 创建滚动文本时记录字体大小

```cpp
void DisplayManager::displayText(...) {
    // ...
    // 设置该行的字体大小（使用当前的全局textSize）
    scrollLines[index].textSize = textSize;
    // ...
}
```

#### 3. 渲染时动态切换字体大小

```cpp
void DisplayManager::loop() {
    for (int i = 0; i < maxLines; i++) {
        if (scrollLines[i].isActive && ...) {
            // 设置该行的字体大小
            dma_display->setTextSize(scrollLines[i].textSize);
            // ... 渲染逻辑 ...

            // 恢复全局字体大小
            dma_display->setTextSize(textSize);
        }
    }
}
```

### 关键点

- `setTextSize()` 只改变全局 `textSize` 变量，不影响已存在的滚动文本
- 新的滚动文本使用当前的 `textSize` 并保存到 `ScrollLine.textSize`
- 渲染时使用每行自己的 `textSize`

## 重叠检测机制

### 静态文本覆盖滚动文本

当显示静态文本时，自动检测并清除被覆盖的滚动文本：

```cpp
void DisplayManager::displayText(...) {
    if (!isScroll) {
        // 计算静态文本的渲染区域
        int startY = yPosition;
        int endY = ...;

        // 检查并清除重叠的滚动文本
        for (int i = 0; i < maxLines; i++) {
            if (scrollLines[i].isActive && ...) {
                int scrollY = scrollLines[i].yPosition;
                int scrollHeight = scrollLines[i].textSize * 16;

                // 检查是否有重叠
                if (!(endY <= scrollY || startY >= scrollY + scrollHeight)) {
                    // 有重叠，清除该滚动文本
                    free(scrollLines[i].content);
                    scrollLines[i].content = nullptr;
                    scrollLines[i].isActive = false;
                    scrollLines[i].isScrolling = false;
                }
            }
        }
    }
}
```

### 滚动文本互相覆盖

当添加新滚动文本时，自动检测并清除被覆盖的其他滚动文本：

```cpp
void DisplayManager::displayText(...) {
    if (isScroll) {
        int yPosition = (line - 1) * 16;
        int lineHeight = textSize * 16;

        // 清除被新滚动文本覆盖的其他滚动文本
        for (int i = 0; i < maxLines; i++) {
            if (i != index && scrollLines[i].isActive && ...) {
                int scrollY = scrollLines[i].yPosition;
                int scrollHeight = scrollLines[i].textSize * 16;

                // 检查是否有重叠
                if (!(yPosition + lineHeight <= scrollY || yPosition >= scrollY + scrollHeight)) {
                    // 有重叠，清除该滚动文本
                    free(scrollLines[i].content);
                    scrollLines[i].content = nullptr;
                    scrollLines[i].isActive = false;
                    scrollLines[i].isScrolling = false;
                }
            }
        }
    }
}
```

### 重叠判断逻辑

两个矩形区域 `[y1, y1+h1)` 和 `[y2, y2+h2)` 不重叠的条件：
```
y1+h1 <= y2  OR  y2+h2 <= y1
```

反之，重叠的条件：
```
NOT (y1+h1 <= y2 OR y2+h2 <= y1)
等价于 (y1+h1 > y2 AND y2+h2 > y1)
```

## 内存管理

### 动态分配

```cpp
// DisplayManager.cpp: displayText()
int len = strlen(textContent) + 1;
scrollLines[index].content = (char *)malloc(len * sizeof(char));
if (scrollLines[index].content != nullptr) {
    strcpy(scrollLines[index].content, textContent);
}
```

### 内存释放

```cpp
// DisplayManager.cpp: clearScrollLine()
void DisplayManager::clearScrollLine(int line) {
    if (line >= 1 && line <= maxLines && scrollLines) {
        int index = line - 1;
        if (scrollLines[index].content != nullptr) {
            free(scrollLines[index].content);
            scrollLines[index].content = nullptr;
        }
        scrollLines[index].isActive = false;
        scrollLines[index].isScrolling = false;
        scrollLines[index].xPosition = PANEL_RES_X;
    }
}

// DisplayManager.cpp: freeAllScrollLines()
void DisplayManager::freeAllScrollLines() {
    if (scrollLines) {
        for (int i = 0; i < maxLines; i++) {
            if (scrollLines[i].content != nullptr) {
                free(scrollLines[i].content);
                scrollLines[i].content = nullptr;
            }
            scrollLines[i].isActive = false;
            scrollLines[i].isScrolling = false;
        }
    }
}
```

### 内存安全

- 在分配新文本前先释放旧文本
- 重叠检测时释放被覆盖的文本
- 全局清屏时释放所有滚动文本
- 析构函数中释放所有内存

## 实现细节

### loop()函数

```cpp
void DisplayManager::loop() {
    // 检查是否有任何滚动行需要更新
    bool hasScrollingLines = false;
    for (int i = 0; i < maxLines; i++) {
        if (scrollLines[i].isActive && scrollLines[i].isScrolling && scrollLines[i].content) {
            hasScrollingLines = true;
            break;
        }
    }

    if (!hasScrollingLines) return;

    // 刷新DMA缓冲区
    dma_display->flipDMABuffer();

    // 更新所有激活的滚动行
    for (int i = 0; i < maxLines; i++) {
        if (scrollLines[i].isActive && scrollLines[i].isScrolling && scrollLines[i].content) {
            unsigned long now = millis();
            if (now > scrollLines[i].lastUpdateTime) {
                int yPosition = scrollLines[i].yPosition;
                int lineHeight = scrollLines[i].textSize * 16;

                // 清除当前行区域
                dma_display->fillRect(0, yPosition, PANEL_RES_X, lineHeight, blackColor);

                // 更新文本位置
                scrollLines[i].xPosition += scrollLines[i].scrollXMove;

                // 检查文本是否完全移出屏幕
                int16_t xOne, yOne;
                uint16_t textWidth, textHeight;
                dma_display->setTextSize(scrollLines[i].textSize);
                dma_display->getTextBounds(scrollLines[i].content, scrollLines[i].xPosition, yPosition,
                                          &xOne, &yOne, &textWidth, &textHeight);

                if (scrollLines[i].scrollDirection == SCROLL_LEFT) {
                    if (scrollLines[i].xPosition + textWidth <= 0) {
                        scrollLines[i].xPosition = PANEL_RES_X;
                    }
                } else {
                    if (scrollLines[i].xPosition >= PANEL_RES_X) {
                        scrollLines[i].xPosition = -textWidth;
                    }
                }

                // 绘制滚动文本
                dma_display->setTextColor(scrollLines[i].color);
                dma_display->setCursor(scrollLines[i].xPosition, yPosition);
                dma_display->printUTF8(scrollLines[i].content);

                // 恢复全局字体大小
                dma_display->setTextSize(textSize);

                // 设置下一次更新时间
                scrollLines[i].lastUpdateTime = now + scrollLines[i].scrollTimeDelay;
            }
        }
    }
}
```

### setTextSize()函数

```cpp
void DisplayManager::setTextSize(int size) {
    // 只设置字体大小，不重新分配滚动行数组
    // 不同字体大小的滚动文本可以同时存在
    if (dma_display == nullptr) return;
    textSize = size;
    dma_display->setTextSize(textSize);
}
```

## 使用示例

### 基础示例：不同字体大小同时滚动

```cpp
#include "DisplayManager.h"

void setup() {
    displayManager.init();

    // 第1行：字体大小1，慢速向左滚动
    displayManager.setTextSize(1);
    displayManager.displayText("Hello World!", true, displayManager.whiteColor, 1, false, 1, DisplayManager::SCROLL_LEFT);

    // 第2行：字体大小2，中速向左滚动
    displayManager.setTextSize(2);
    displayManager.displayText("大字体文本", true, displayManager.redColor, 2, false, 2, DisplayManager::SCROLL_LEFT);
}

void loop() {
    displayManager.loop();
}
```

### 高级示例：不同速度和方向

```cpp
void setup() {
    displayManager.init();

    // 第1行：字体1，快速向左滚动
    displayManager.setTextSize(1);
    displayManager.displayText("Fast Scroll Left", true, displayManager.greenColor, 1, false, 3, DisplayManager::SCROLL_LEFT);

    // 第2行：字体1，慢速向右滚动
    displayManager.setTextSize(1);
    displayManager.displayText("Slow Scroll Right", true, displayManager.blueColor, 2, false, 1, DisplayManager::SCROLL_RIGHT);

    // 第3行：字体2，中速向左滚动
    displayManager.setTextSize(2);
    displayManager.displayText("中速大字体", true, displayManager.yellowColor, 3, false, 2, DisplayManager::SCROLL_LEFT);
}

void loop() {
    displayManager.loop();
}
```

### 动态修改示例

```cpp
void setup() {
    displayManager.init();

    // 初始显示
    displayManager.setTextSize(1);
    displayManager.displayText("Line 1", true, displayManager.whiteColor, 1);
    displayManager.setTextSize(2);
    displayManager.displayText("Line 2", true, displayManager.redColor, 2);
}

void loop() {
    displayManager.loop();

    // 每5秒修改第2行的速度
    static unsigned long lastChange = 0;
    if (millis() - lastChange > 5000) {
        static int speed = 1;
        speed = (speed % 3) + 1;
        displayManager.setLineScrollSpeed(2, speed);
        lastChange = millis();
    }
}
```

## 注意事项

### 1. 静态文本与滚动文本冲突

当有滚动文本存在时，静态文本的`autoWrap`参数会被强制设为`false`，以避免冲突：

```cpp
// 如果有滚动文本，强制 autoWrap 为 false
bool hasScrolling = false;
for (int i = 0; i < maxLines; i++) {
    if (scrollLines[i].isActive && scrollLines[i].isScrolling) {
        hasScrolling = true;
        break;
    }
}
if (hasScrolling) {
    autoWrap = false;
}
```

### 2. 行号范围

- 滚动文本：`line`必须在 `1` 到 `maxLines` 范围内
- 静态文本：`line <= 0`（包括-1）时清屏并全屏显示；`line > 0`时在指定行显示

### 3. 字体大小与覆盖

大字体会覆盖多行区域：

| 行号 | 字体1 | 字体2 | 字体3 | 字体4 |
|------|-------|-------|-------|-------|
| 1 | ✓ | ✓ | ✓ | ✓ |
| 2 | ✓ | ✓ | ✓ | ✓ |
| 3 | ✓ | - | ✓ | ✓ |
| 4 | ✓ | - | - | ✓ |

- ✓表示该行被覆盖
- `-`表示未被覆盖

### 4. 内存限制

- 每行文本动态分配内存，应避免过长的文本
- 系统会自动清理被覆盖的文本
- 使用`clearScrollLine()`或`clearAll()`手动清理

### 5. 屏幕分辨率适配

修改`PANEL_RES_Y`后，最大行数自动计算：

```cpp
// config.h
#define PANEL_RES_Y 32  // 64x32屏幕，支持2行
// 或
#define PANEL_RES_Y 64  // 64x64屏幕，支持4行
```

## 总结

多行滚动文本功能通过以下核心设计实现：

1. **动态最大行数**: `maxLines = PANEL_RES_Y / 16`
2. **独立字体记录**: 每行`ScrollLine`保存自己的`textSize`
3. **固定Y坐标系统**: `(line-1) * 16` 简化坐标计算
4. **动态字体切换**: 渲染时根据每行的`textSize`切换字体
5. **重叠检测机制**: 自动清除被覆盖的文本
6. **动态内存管理**: 使用`malloc/free`管理文本内容

这些设计使得不同字体大小、速度、方向的文本可以互不干扰地同时滚动，并且能够自动适配不同的屏幕分辨率。
